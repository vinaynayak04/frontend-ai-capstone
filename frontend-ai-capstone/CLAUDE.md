# Claude Instructions

## Project Overview

This repository contains assignments and projects for the Frontend AI Engineering track. The goal is to build modern, maintainable web applications while leveraging AI-assisted development.

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Node.js
- Git & GitHub
- VS Code / Cursor
- Claude Code

---

## Coding Conventions

- Use meaningful variable and function names.
- Write clean, readable, and modular code.
- Keep functions small and focused on a single responsibility.
- Add comments only where necessary.
- Use consistent formatting throughout the project.

---

## Git Workflow

Follow Conventional Commits for all commits.

Examples:

- feat: add login page
- fix: resolve navbar alignment issue
- docs: update README
- style: improve code formatting
- refactor: simplify utility functions
- chore: update dependencies

---

## AI Usage Guidelines

Claude Code may be used to:

- Explain unfamiliar concepts
- Generate boilerplate code
- Review existing code
- Suggest improvements
- Debug errors
- Improve documentation

All AI-generated code should be reviewed and tested before committing.
